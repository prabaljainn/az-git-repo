Content

HTML

HTML stands for HyperText Markup Language. It's used to structure content on the web.

CSS

CSS stands for Cascading Style Sheets. It's used to style the structured content created by HTML.

JavaScript

JavaScript is a programming language that enables interactive web pages and thus is a part of most web browsers.

Web Technologies

Technology Type Description
